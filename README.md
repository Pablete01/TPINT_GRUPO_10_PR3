Este es un proyecto desarrollado para la materia Programacion III de la carrera Tecnicatura Universitaria en Programacion.
El mismo es un software de gestion para una clinica. Tiene acceso como adminitrador y como medico.
El sistema permite getionar medicos (ABML), pacientes (ABML) y turnos. Ademas genera informes.

Desarrollado en .NET y C#
